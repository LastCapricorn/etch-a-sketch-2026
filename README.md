# Etch - A - Sketch - 2026

A very entertaining Project from
[The Odin Project](https://www.theodinproject.com)'s Foundations Course.
My first contact with TOP was in 2019 and I chose the *Ruby Path* at that time ...and somehow got lost. In 2023 I started again, now on the ***JavaScript Path*** , much better but I got stuck again.

Surprisingly, despite my 3 years of coding abstinence, this version now is (in my opinion) clearly better, with more functionality but about 130 lines less JS-code than the one from 2023.

I'm very curious to see how long I will be able to stick to the path this time and which progress I will experience with the upcoming TOP-projects.

## The Features this App provides, are the following

- **Light and Dark Theme**

  - the *"art work"* remains preserved
  - only unchanged blocks will be inverted

- **Resize Sketch Board**

  - ranges from 16 x 16 to 96 x 96 with steps of 16

- **Clear the Board**

- **Toggle Grid On and Off**

- **Seven *Preconfigured* Drawing Modes**

  - those which are needed to fulfill the TOP-Assignment and the ones for extra credit
  - and some more, because I had lots of fun
  - additional color picker for anybodies preferred color

- **Using Mouse Buttons**

  - left for drawing, right to erase
  - except luminance mode: left is darkening, right is brightening

- **And Last but not Least:**

  - **NO** CSS Grid - as requested
